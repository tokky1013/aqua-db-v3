Songクラス、Dataクラスに持たせる機能
・Jsonからデータを読み込む
　読み込みは、最初にGETでデータを取得するときとPOSTしてレスポンスが帰ってきた時
・POSTを送ってスプレッドシートのデータ更新

Data
・uuid
・createdAt
・songId
・key
・score
・machineType
・comment
・hasSung

Song
・uuid
・createdAt
・title
・artist
・chestMinNote（地声最低音）
・chestMaxNote（地声最高音）
・headMinNote（裏声最低音）
・headMaxNote（裏声最高音）
・overallMaxNote（全体の最高音）

curl 例
・GET data
curl "https://script.google.com/macros/s/AKfycbxViVZWS3378lxd9DSF1qYsEiXWoYM3yV-fSo3oQu1wsnpGwCZwr0xIKmRVRLjPc7us/exec?token=d47f6f90-db00-4208-a56b-0c3640837f33&sheet=data"

・POST data add
curl "https://script.google.com/macros/s/AKfycbxViVZWS3378lxd9DSF1qYsEiXWoYM3yV-fSo3oQu1wsnpGwCZwr0xIKmRVRLjPc7us/exec" -d "token=d47f6f90-db00-4208-a56b-0c3640837f33" -d "sheet=data" -d "mode=add" -d "uuid=aaa" -d "createdAt=2" -d "songId=bbb" -d "key=3" -d "score=" -d "machineType=" -d "comment=aaa" -d "hasSung=true"

・POST songs delete
curl "https://script.google.com/macros/s/AKfycbxViVZWS3378lxd9DSF1qYsEiXWoYM3yV-fSo3oQu1wsnpGwCZwr0xIKmRVRLjPc7us/exec" -d "token=d47f6f90-db00-4208-a56b-0c3640837f33" -d "sheet=songs" -d "mode=delete" -d "uuid=325eb4f9-a57f-4af6-81ac-f89e0ffb5db8"
